# ANT_FreeDoc

## Purpose
Exemple of doctors SQL database with a Ruby on Rails app

## Installation
  * Begin to install gems with a classic :
```bundle install```
  * To reset the database execute the command :   
 ```rake db:drop db:create db:migrate``` 
  * then create classes
 ```rails db:migrate ```
  * and populate the database
 ```rails db:seed```
